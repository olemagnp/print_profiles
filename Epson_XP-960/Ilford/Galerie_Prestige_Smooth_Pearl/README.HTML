Target generation

Generate target information files with

targen -v2 -d2 -G -e8 -g128 -f675 -t ilford_gp_smooth_pearl_xp_960_targen

Convert the ti1/ti2-file to a TIFF-target with printtarg

printtarg -v2 -ii1 -a0.89 -b -T500 -m0 -M0 -P -p210x297 ilford_gp_smooth_pearl_xp_960_targen

Open the resulting TIFF in an image editor (e.g. GIMP), crop the margins, and scale to A4-size
See the cropped image for an example

Target printing

Print the generated target with no color management.
